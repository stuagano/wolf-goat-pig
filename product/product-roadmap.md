# Product Roadmap

## Sections

### 1. Game Setup
Create a new game, add players, configure betting rules and starting parameters.

### 2. Live Scorekeeper
The core hole-by-hole interface for tracking shots, captains, partnerships, and betting actions during a round.

### 3. Standings & Settlement
Real-time leaderboard showing who's up/down, and final settlement showing who owes what.

### 4. Game History
Browse completed games, view detailed scorecards, and track performance over time.
