# Wolf Goat Pig MVP

## Backend (FastAPI + SQLite)
- `cd backend`
- `python3 -m venv venv && source venv/bin/activate`
- `pip install -r requirements.txt`
- `uvicorn app.main:app --reload`

## Frontend (React)
- `cd frontend`
- `npm install`
- `npm start`

## Deployment
- Backend: Render.com (free)
- Frontend: Vercel.com or Netlify.com (free)
